
  # AI Nail Try-On App UI (Copy)

  This is a code bundle for AI Nail Try-On App UI (Copy). The original project is available at https://www.figma.com/design/NMwKQJCoS2jELowb9fLR4G/AI-Nail-Try-On-App-UI--Copy-.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Backend integration

  The app calls the FastAPI backend through `VITE_API_BASE_URL`.

  ```bash
  cp .env.example .env
  npm install
  npm run dev
  ```

  Default backend URL: `http://localhost:8000`.
